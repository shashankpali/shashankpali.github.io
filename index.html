EZYGradientView
"Create gradients and blur gradients without a single line of code !"
"EZYGradientView "exposes five properties you can modify to customize your gradient. The view itself is "IBDesignable" and its properties "IBInspectable" which means you can customize your gradient in real time on an "xib" or "storyboard". Just put a "UIView", assign class, place value and we get above gradient.
"EZYGradientView" placed in top 10 trending repositories on "GitHub"! 
ATZAlertController
ATZAlertController helps you to create an alert from any class without any dependency on ViewController. The name itself derives that it provides A to Z solution towards the UIAlertController with various types of functions you can use them accordingly. ATZAlertController is easy to use and implement.
